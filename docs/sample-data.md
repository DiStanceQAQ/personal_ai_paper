# Sample Data

This repository intentionally does not include third-party research PDFs.

To test the app locally:

1. Create or download PDFs that you have permission to process.
2. Import them through the desktop app or API.
3. Keep local PDFs under `app-data/`, `reference_paper/`, or another ignored
   directory.

If you add sample files for tests or demos, prefer synthetic PDFs generated by
test code or documents with an explicit redistribution license.
