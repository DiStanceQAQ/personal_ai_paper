## Codebase Patterns
- Python 3.12 with FastAPI + uvicorn for backend, static HTML for frontend
- `uv venv` for virtual environment management, `uv pip install` for dependencies
- `mypy` with `strict = true` in pyproject.toml
- `pytest` with `asyncio_mode = "auto"` and `httpx.ASGITransport` for FastAPI testing
- `Makefile` provides `dev`, `test`, `typecheck`, and `check` targets
- All source files under project root; tests in `tests/` directory
- Static files served from `static/` via FastAPI `StaticFiles` mount
- Use `pip install -e ".[dev]"` for editable install with dev dependencies
